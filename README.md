
# Task Management System Frontend

Frontend application for the Task Management System built using React, TypeScript, Tailwind CSS, and shadcn/ui.

## Tech Stack

- React.js
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- Axios
- React Router DOM
- React Hook Form
- Zod
- Lucide React

---

## Features

- User authentication UI
- JWT token handling
- Task listing interface
- Create task form
- Update task functionality
- Delete task functionality
- Task priority and status management
- Responsive UI
- Loading and error states
- API integration with Django backend
